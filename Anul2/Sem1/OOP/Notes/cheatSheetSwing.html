1. JFrame (Fereastra Principală)

Descriere:

JFrame este containerul de bază pentru aplicațiile GUI. Acesta servește drept fereastră principală.

Metode Importante:

  • setTitle(String title) – Setează titlul ferestrei.
  • setSize(int width, int height) – Dimensiunile ferestrei.
  • setDefaultCloseOperation(int operation) – Comportamentul la închidere (ex. EXIT_ON_CLOSE).
  • setVisible(boolean visible) – Afișează sau ascunde fereastra.
  • add(Component comp) – Adaugă componente în fereastră.

Exemplu:

import javax.swing.*;

public class JFrameExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Exemplu JFrame");
        frame.setSize(300, 200);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setLayout(new FlowLayout());

        frame.add(new JButton("Buton 1"));
        frame.add(new JLabel("Salut, lume!"));

        frame.setVisible(true);
    }
}

2. JPanel (Container Secundar)

Descriere:

JPanel este folosit pentru organizarea altor componente într-o fereastră.

Metode Importante:

  • setLayout(LayoutManager layout) – Specifică layout-ul pentru componente.
  • setBackground(Color color) – Schimbă culoarea de fundal.
  • add(Component comp) – Adaugă componente.
  • paintComponent(Graphics g) – Suprascris pentru a desena grafica personalizată.

Exemplu:

import javax.swing.*;
import java.awt.*;

public class JPanelExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Exemplu JPanel");
        JPanel panel = new JPanel();
        panel.setLayout(new GridLayout(2, 2));
        panel.setBackground(Color.LIGHT_GRAY);

        panel.add(new JButton("1"));
        panel.add(new JButton("2"));
        panel.add(new JButton("3"));
        panel.add(new JButton("4"));

        frame.add(panel);
        frame.setSize(300, 200);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}

3. JButton (Buton)

Descriere:

JButton este un buton care declanșează evenimente când este apăsat.

Metode Importante:

  • setText(String text) – Setează textul butonului.
  • addActionListener(ActionListener listener) – Adaugă un ascultător de eveniment pentru acțiuni.
  • setEnabled(boolean enabled) – Activează/dezactivează butonul.

Exemplu:

import javax.swing.*;
import java.awt.event.*;

public class JButtonExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Exemplu JButton");
        JButton button = new JButton("Apasă-mă");

        button.addActionListener(new ActionListener() {
            public void actionPerformed(ActionEvent e) {
                JOptionPane.showMessageDialog(null, "Buton apăsat!");
            }
        });

        frame.add(button);
        frame.setSize(200, 100);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}

4. JLabel (Etichetă)

Metode Importante:

  • setText(String text) – Schimbă textul.
  • setIcon(Icon icon) – Adaugă o imagine.
  • setHorizontalAlignment(int alignment) – Alinierea orizontală (CENTER, LEFT, RIGHT).

Exemplu:

import javax.swing.*;

public class JLabelExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Exemplu JLabel");
        JLabel label = new JLabel("Etichetă Centrată");
        label.setHorizontalAlignment(SwingConstants.CENTER);

        frame.add(label);
        frame.setSize(300, 100);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}

5. JTextField (Câmp de Text)

Metode Importante:

  • setText(String text) – Setează textul.
  • getText() – Obține textul.
  • setColumns(int columns) – Lățimea câmpului.

Exemplu:

import javax.swing.*;

public class JTextFieldExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Exemplu JTextField");
        JTextField textField = new JTextField(10);

        JButton button = new JButton("Afișează");
        button.addActionListener(e -> {
            JOptionPane.showMessageDialog(null, "Text: " + textField.getText());
        });

        JPanel panel = new JPanel();
        panel.add(new JLabel("Introdu text:"));
        panel.add(textField);
        panel.add(button);

        frame.add(panel);
        frame.setSize(300, 150);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}

6. MouseListener și MouseMotionListener

MouseListener Metode Importante:

  • mouseClicked(MouseEvent e) – Detectează click-uri.
  • mousePressed(MouseEvent e) – Detectează apăsarea butonului mouse.
  • mouseReleased(MouseEvent e) – Detectează eliberarea butonului mouse.
  • mouseEntered(MouseEvent e) – Mouse intră într-o componentă.
  • mouseExited(MouseEvent e) – Mouse iese dintr-o componentă.

Exemplu MouseListener:

import javax.swing.*;
import java.awt.event.*;

public class MouseListenerExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("MouseListener Example");
        JPanel panel = new JPanel();

        panel.addMouseListener(new MouseAdapter() {
            public void mouseClicked(MouseEvent e) {
                System.out.println("Click la: " + e.getX() + ", " + e.getY());
            }

            public void mouseEntered(MouseEvent e) {
                panel.setBackground(java.awt.Color.YELLOW);
            }

            public void mouseExited(MouseEvent e) {
                panel.setBackground(java.awt.Color.LIGHT_GRAY);
            }
        });

        panel.setBackground(java.awt.Color.LIGHT_GRAY);
        frame.add(panel);
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}

7. Timer (Animație)

Metode Importante:

  • start() – Pornește animația.
  • stop() – Oprește animația.

Exemplu:

import javax.swing.*;
import java.awt.*;
import java.awt.event.*;

public class TimerExample extends JPanel implements ActionListener {
    int x = 10, y = 100;
    Timer timer = new Timer(50, this);

    public TimerExample() {
        timer.start();
    }

    public void paintComponent(Graphics g) {
        super.paintComponent(g);
        g.fillOval(x, y, 30, 30);
    }

    public void actionPerformed(ActionEvent e) {
        x += 5;
        if (x > getWidth()) x = 10;
        repaint();
    }

    public static void main(String[] args) {
        JFrame frame = new JFrame("Animație Timer");
        frame.add(new TimerExample());
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}

8. Exemplu Complet de Arhitectură MVC

Arhitectura Model-View-Controller (MVC) oferă o modalitate organizată de separare a logicii aplicației (Model), interfeței grafice (View), și interacțiunii utilizatorului (Controller).

Componentele MVC:

  • Model: Gestionează datele și logica aplicației.
  • View: Afișează datele din Model către utilizator.
  • Controller: Preia intrările utilizatorului și actualizează Modelul și View-ul corespunzător.

Exemplu Practic: Calculator Simplu

Acest exemplu implementează un calculator simplu care realizează adunări și resetări, utilizând arhitectura MVC.

Structura Codului

  1. Model – Gestionează logica matematică.
  2. View – Interfața grafică (GUI).
  3. Controller – Ascultătorii care tratează evenimentele din View.

Codul Complet

1. Model (CalcModel.java)

public class CalcModel {
    private int total;  // Stochează rezultatul

    public CalcModel() {
        total = 0;  // Valoare inițială
    }

    public void add(int value) {
        total += value;  // Adaugă valoarea
    }

    public void reset() {
        total = 0;  // Resetează rezultatul
    }

    public int getTotal() {
        return total;  // Returnează rezultatul
    }
}

2. View (CalcView.java)

import javax.swing.*;
import java.awt.*;

public class CalcView extends JFrame {
    private JTextField inputField = new JTextField(5);
    private JButton addButton = new JButton("Adună");
    private JButton resetButton = new JButton("Reset");
    private JLabel resultLabel = new JLabel("Total: 0");

    public CalcView() {
        setTitle("Calculator Simplu - MVC");
        setLayout(new FlowLayout());
        setSize(300, 150);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        add(new JLabel("Introduceți un număr:"));
        add(inputField);
        add(addButton);
        add(resetButton);
        add(resultLabel);

        setVisible(true);
    }

    public String getInput() {
        return inputField.getText();  // Returnează textul introdus
    }

    public void setTotal(int total) {
        resultLabel.setText("Total: " + total);  // Actualizează rezultatul
    }

    public void addAddButtonListener(java.awt.event.ActionListener listener) {
        addButton.addActionListener(listener);  // Ascultător pentru butonul "Adună"
    }

    public void addResetButtonListener(java.awt.event.ActionListener listener) {
        resetButton.addActionListener(listener);  // Ascultător pentru butonul "Reset"
    }
}

3. Controller (CalcController.java)

import java.awt.event.*;

public class CalcController {
    private CalcModel model;
    private CalcView view;

    public CalcController(CalcModel model, CalcView view) {
        this.model = model;
        this.view = view;

        // Adăugăm ascultători pentru butoane
        this.view.addAddButtonListener(new AddButtonListener());
        this.view.addResetButtonListener(new ResetButtonListener());
    }

    // Ascultător pentru butonul "Adună"
    class AddButtonListener implements ActionListener {
        public void actionPerformed(ActionEvent e) {
            try {
                int value = Integer.parseInt(view.getInput());
                model.add(value);
                view.setTotal(model.getTotal());
            } catch (NumberFormatException ex) {
                JOptionPane.showMessageDialog(null, "Introduceți un număr valid!");
            }
        }
    }

    // Ascultător pentru butonul "Reset"
    class ResetButtonListener implements ActionListener {
        public void actionPerformed(ActionEvent e) {
            model.reset();
            view.setTotal(model.getTotal());
        }
    }
}

4. Program Principal (CalcMVC.java)

public class CalcMVC {
    public static void main(String[] args) {
        CalcModel model = new CalcModel();  // Creează Modelul
        CalcView view = new CalcView();     // Creează View-ul
        new CalcController(model, view);    // Inițializează Controller-ul
    }
}

Explicație Cod:

  1. Model (CalcModel) gestionează totalul și metodele de adunare și resetare.
  2. View (CalcView) oferă interfața grafică cu câmp de text, butoane, și etichete.
  3. Controller (CalcController) gestionează interacțiunile dintre utilizator și aplicație:
    • Butonul Adună preia numărul din câmpul de text, îl adaugă în Model și actualizează View-ul.
    • Butonul Reset resetează totalul în Model și actualizează View-ul.

Rezultat Final:

  1. Interfață Simplă cu:
    • Câmp de text pentru introducerea numerelor.
    • Butoane „Adună” și „Reset”.
    • Etichetă care afișează totalul curent.
  2. Logica aplicației este separată clar conform arhitecturii MVC.

Beneficii MVC:

  • Separare logică între date, interfață și funcționalități.
  • Cod ușor de întreținut și extins.
  • Permite reutilizarea Modelului în alte aplicații (ex. consolă, web).
